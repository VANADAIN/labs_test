## Quickstart 

Note: Forge (foundry) must be installed. 

[Foundry Link](https://book.getfoundry.sh/getting-started/installation)

To run tests:

```
git clone https://github.com/VANADAIN/labs_test.git
cd labs_test
forge install 
forge test -vv 
forge test -vvvvv // high verbose mode
```
